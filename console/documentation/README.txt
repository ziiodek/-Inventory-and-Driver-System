URL Convention

URL variables:
layout
    Especifies the layout that is gonna be shown to the user i.e grid, form, control_panel
model
    Especifies the model that will bi display to the user i.e Shipping, drivers, trailers
action
    Especifies the action that the user will do with the respected model i.e Add, update, view, delete

URL Convention Example    
www.houndproject.com/index.php?layout=form&model=shipping&action=add
